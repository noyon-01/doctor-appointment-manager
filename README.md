# Project Name: DocAppoint - Doctor Appointment Management System

🔗 **Live Site URL:** https://doctor-appointment-manager-59mv.vercel.app

## 📌 Project Overview
DocAppoint is a modern doctor appointment booking platform built with Next.js, designed to simplify healthcare scheduling for patients and doctors. Users can browse doctors, book appointments, manage profiles, and track booking history seamlessly.

---

## ✨ Features

- 👨‍⚕️ **Browse Doctors** – View all available doctors with detailed profiles and specialties.
- 📅 **Book Appointments** – Easy and fast appointment booking system with real-time availability.
- 👤 **User Dashboard** – Manage profile information and view all booked appointments.
- 🔐 **Authentication System** – Secure login and registration using Better Auth.
- 📱 **Responsive Design** – Fully responsive UI for mobile, tablet, and desktop devices.
- ⚡ **Fast Performance** – Built with Next.js for optimized speed and SEO.
- 🎨 **Modern UI/UX** – Clean and user-friendly interface for better user experience.

---

## 🛠️ Technologies Used

- Next.js 16
- Tailwind CSS
